# demoexam
Готовый продукт по ТЗ ДемоЭкзамена (pdf документ стр 29)
# Как использовать?
1) Импортируйте базу данных (demoexam)
2) Закиньте файлы проекта к себе на хост
3) В файле includes/db/config.php нужно изменить данные от БД
4) Готово!
